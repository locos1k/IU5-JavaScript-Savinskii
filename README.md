## Задание

Часть 1. Создание структуры страницы
1. Создать HTML-файл calculator.html.
2. Описать базовую структуру HTML-документа:
* объявление ```<!DOCTYPE html>```;
* теги ```<html>, <head>, <body>```;
* указание кодировки и заголовка страницы.
3. Сверстать структуру калькулятора:
* контейнер;
* блок вывода результата;
* кнопки цифр (0–9, точка);
* кнопки операций (C, +/-, %, /, x, -, +, =).
4. Для каждого интерактивного элемента задать уникальный id.
5. Использовать контейнеры ```<div>``` для группировки строк кнопок.

Часть 2. Подключение и использование CSS
1. Создать отдельный файл стилей style.css.
2. Подключить файл к HTML-документу через ```<link rel="stylesheet">```.
3. Реализовать стили:
* базовый стиль кнопок;
* стили для кнопок операций (основные и второстепенные);
* стиль кнопки “=”;
* стиль блока вывода результата.
4. Применить:
* селекторы по классу;
* селекторы по идентификатору;
* псевдоклассы :hover и :active.
5. Добиться корректного отображения калькулятора в браузере.
